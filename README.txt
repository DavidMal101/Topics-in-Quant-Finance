# Topics-in-Quant-Finance

This repository contains the coursework solutions for the module Selected Topics in Quantitative Fianance ICL. The authors are David Malone, Pablo, Gómez Hernández and Alberto Moreno de Vega García.
The course deals with a new approach to simulate arbitrage free implied vol surfaces using VolGAN: Cont, Rama and Vuletić, Milena, Simulation of Arbitrage-Free Implied Volatility Surfaces (December 11, 2022). Available at SSRN: https://ssrn.com/abstract=4299363 or http://dx.doi.org/10.2139/ssrn.4299363
